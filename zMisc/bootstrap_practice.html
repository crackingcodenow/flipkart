CSS & BS Practice:

Position : static (normal)

left : 10px;

top : 10px;

right : 0;

This is a static element
which means it doesn't
move even after the styles are
applied on the page.

Position : fixed;

top : 1;

right : 0;

This element is positioned
relative to the viewport
which means it doesn't
move with the flow of the page.

Position : Relative

left : 10;

top : 10;

This element is positioned
relative to its normal postion,
that means the actual position
when relative is not applied.

Position : Relative;

width : 400px;

height : 280px;

Position : Absolute;

top : 80px;
right : 0;

width: 230px;
height : 180px;

Position : Sticky

left : 200px;

top : 10px;

This element is positioned
is positioned based on the user's scroll position.
It toggles between relative and fixed, depending
on the scroll position.
It is positioned relative until a
given offset position is met in the viewport -
then it "sticks" in place (like position:fixed).
