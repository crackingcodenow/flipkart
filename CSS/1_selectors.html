4.DESCENDENT Selector:

There are many DESCENDENT selectors : (ul-li, ol-li), (datalist-option, select-option), (thead,tbody,tfoot - tr), (tr-td),

Syntax:

< head >
  < style >

    parent > child {
    attribute : value ;
    attribute : value ;
    }

  < /style >
< /head >

< body >
  < ul >
   < li > HTML < /li >
   < li > CSS < /li >
  < /ul >
< /body >

5.ATTRIBUTE Selector:

There are many ATTRIBUTE selectors : input[type=text / button / etc],   img[width=300],   aside[width=300],

Syntax:

< head >
  < style >

    tagname [attribute = value] {
       styleAttribute : value;
    }

  < /style >
< /head >

< body >
  < input type="text" >
  < input type="button" value="Submit">
< /body >
