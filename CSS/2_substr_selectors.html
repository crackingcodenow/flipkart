SUB-STR STARTS / ENDS / ANYWHERE WITH (^$*=):

Syntax:

  < style >

    E[attr ^= value], E[attr $= value], E[attr *= value] {

    color : white;
    }

  < /style >
< /head >

Attribute Selectors with SPACES / SPCL chars (~|=):

It is used when the attributes defined are having spaces or hyphens in between the same word.
Spaces and hyphens here doesn't mean they are two seperate words as in Class selectors but are treated as a single attr selector.

Syntax:

< head >
  < style >

    E[attr ~= value], E[attr |= value] {
    background-color : blueviolet;
    color : white;
    }

  < /style >
< /head >

< body >
  < h2 class ~= "text effects" > HTML < /h2 >
  < h2 class |= "text-effects" > HTML < /h2 >
< /body >
